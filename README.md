# single-chain-xujin-orchestrator
All-in-one multi-agent workflow tool. Generate single-chain agent workflow config from prompts, built-in lightweight self-developed scheduling engine. Equipped with multi-layer deliverable gate checks, retry/fail exception strategy, no freeze mechanism. 一体化多Agent工作流工具，提示词驱动生成单链Agent配置，内置轻量化自研调度引擎，内置多层交付物门禁，采用重试/失败二分异常策略。
